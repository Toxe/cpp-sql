# C++ SQL Examples

[![CI](https://github.com/Toxe/cpp-sql/workflows/CI/badge.svg)](https://github.com/Toxe/cpp-sql/actions)

## Dependencies

- CMake
- sqlpp11
- sqlpp11-connector-mysql
- [JSON for Modern C++](https://github.com/nlohmann/json)

### Vcpkg

Install Vcpkg dependencies with:

    $ vcpkg install $(< .vcpkg)
